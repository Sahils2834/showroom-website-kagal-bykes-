# 🏍️ Kagal Bykes - Showroom Website

A comprehensive, modern, and impressive showroom website for Hero MotorCorp authorized dealers. Built with React, Vite, Tailwind CSS, and Node.js backend.

## 📋 Website Overview

**Kagal Bykes** is a fully-featured bike showroom website with professional design, impressive UI/UX, and all necessary functionalities for a successful showroom presence online.

## 🚀 Getting Started

### Installation:

```bash
# Backend setup
cd backend
npm install
npm start

# Frontend setup (new terminal)
cd frontend
npm install
npm run dev
```

### Access:
- **Frontend**: http://localhost:5173
- **Backend**: http://localhost:5000

## 📝 Customization Guide

### To Change Showroom Name:
Edit `src/components/Navbar.jsx` and `src/components/Footer.jsx`

### To Add/Edit Bikes:
Edit `src/data/bikesData.js` - add to `bikesData` array

### To Modify Services:
Edit `src/data/bikesData.js` - modify `serviceCategories` array

### To Update Finance Options:
Edit `src/data/bikesData.js` - modify `financeOptions` array

### To Change Colors:
Modify Tailwind classes in components or update `tailwind.config.js`

### To Add Reviews:
Edit `src/data/bikesData.js` - modify `reviewsData` array
